## 5. Lessons & Reusable Insights
1. When MCP tool handlers receive array parameters, they may arrive as JSON strings — always add defensive parsing at the handler boundary.
2. MCP servers in Claude Code don't hot-reload. `claude mcp remove` + `claude mcp add` updates config but the running process persists until a new session starts.
3. When an MCP tool is broken mid-session, write a standalone script using the same auth/config to accomplish the task rather than waiting for a restart.
4. `claude mcp add` without `--scope user` defaults to project scope — use `--scope user` for globally available servers.
5. When pushing to a repo you don't own, check `git remote -v` first — fork via `gh repo fork` then `git remote set-url origin` to your fork.
6. Gmail `modifyMessageLabels` API requires `addLabelIds` as a proper array of strings — passing a stringified array causes "Invalid label" errors.
